Disposan Bestellungen über File Transfer Protocoll

Bezeichnung: Disposan Bestellungen über File Transfer Protocoll
Author: Marco Descher <descher@medevit.at> 

Version: 1.0.1, Februar 2013
Beschreibung: Dieses Plugin ermöglicht den automatischen Bestellungstransfer mittels FTP an die Firma Disposan.

Anforderungen

Diese Plug-In benötigt Elexis in Version >2.1.7.

Installation

Die Installation des Plug-Ins selbst erfolgt mittels des Konfigurators, es müssen keinerlei spezielle Massnahmen ergriffen werden.

Konfiguration

Das Plug-In muss für die Verwendung in der Einstellungsseite konfiguriert werden, siehe Grafik.

Hier müssen

  • Adresse des FTP Servers
  • Unterverzeichnis (falls notwendig)
  • Benutzername
  • Passwort
  • Kundennummer
angegeben werden. Diese Daten sind erforderlich um eine erfolgreiche Übermittlung durchzuführen.
