Allgemeines

Für das Verschicken von Rechnungen an Medidata gibt es zwei plugins:

  • Das alte, von der MedShare 2015 als OpenSource veröffentliche Mediport-Plugin
  • Das neuere, propritäre von der Medelexis Medidata-Plugin

Um Rechnungen an die Medidata verschicken zu können, muss jede Praxis von Medidata eine Lizenz bekommen.

Die Medidata vergibt für Elexis nur Lizenzen an von Medelexis anerkannte OC (Operation Center). Eine Liste der Medelexis Supportpartner finden Sie unter https://www.medelexis.ch/index.php?id=121.

Die Medelexis Operation Center sind vertraglich verpflichtet, Support nur an Arztpraxen zu leisten, welche Kunden von Medelexis sind.

Lizenzen für Benutzer der freien Elexis-Variante

Mitte 2016 wurde vereinbart, dass

  • die Medelexis auf Anfrage an Julian Kappis (julian - at - medelexis.ch) der Medidata das Okay gibt, einer Arztpraxis, welche die frei erhältliche Elexis-OpenSource Variante einsetzt, eine Lizenz zu erteilen.
  • Arztpraxis, welche die frei erhältliche Elexis-OpenSource Variante einsetzen, können von Niklaus Giger (niklaus.giger -at- member.fsf.org), welche für die Medelexis AG als Release-Manager tätig ist, bezahlten Support zu branchenüblichen Tarifen für das Mediport Plugin erhalten

Installation & Konfiguration des Mediport Communicators

Dies ist eine Arbeit, welche im Normall ein OC oder der Informatik-Unterstützer macht.

Die manuelle Installation für Windows ist unter Medidata MPC Installation dokumentiert.

Eine automatische Installation für Linux-Server ist unter [Salt für MPC] dokumentiert.
