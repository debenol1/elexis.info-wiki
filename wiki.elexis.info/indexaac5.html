Release 3.5.0 - 27.3.2018

In der neuen Version wurden Fehler aus Elexis 3.4.0 behoben und es gibt diverse neue Features.

In der untenstehenden Liste beziehen sich die Zahlen in Klammern auf das nicht öffentliche Redmine-Ticket-System der Medelexis AG.

Teile, welche nur Medelexis-Kunden betreffen, sollten in dieser Beschreibung immer wie folgt markiert sein: Proprietär: Exklusiv für Medelexis Kunden

Upgrade

  • Beachten Sie bitte die unten beschriebenen Voraussetzungen
  • Wir empfehlen alles zuerst auf einer Test-Datenbank durchzuspielen und zu testen. Sie könne dafür z.B. einen MySQL Docker Container gebrauchen
    • Zumindest ein Datenbank-Backup wird vor dem Update empfohlen
  • Danach müssen wie aus einer der 3 Möglichkeinte Elexis auf 3.5 hochziehen
    • Automatischer Upgrade (nur für Medelexis-Kunden)
    • Elexis OpenSource 3.5.0 neu installieren
    • Elexis Kern und Features via neue P2-Seite aktualisieren

Voraussetzungen

Elexis 3.x läuft nicht mit Java 9. Wahrscheinlich mit Java 1.7. Wir empfehlen Java 8 (OpenJDK oder Sun/Oracle).

Vorgängig muss eine Version 3.1.x installiert sein.

Der MySQL-Server muss mindestens auf der Version 5.7 sein - unter dieser Version werden die neu genutzten SQL-Views nicht unterstützg. Gemäss https://dev.mysql.com/doc/refman/5.7/en/upgrading.html muss man von 5.5 in zwei Schritten (via 5.6) auf 5.7 upgraden. Jede andere Methode als mysqldump/restore ist nicht praktikabel.

Elexis Kern und Features via neue P2-Seite aktualisieren

  • Elexis auf 3.4.0 aktualisieren über "Hilfe > Auf Updates überprüfen"
  • Menü "Hilfe > Neue Software installieren" anwählen. Link "Work with the list of selected software sites" wählen, dort
    • Eintrag "Elexis 3 Core release" mit "Edit" bearbeiten, Name auf "Elexis 3 Core 3.5" und Location auf "http://download.elexis.info/elexis/3.5/p2/elexis-3-core" setzen. Mit "OK" bestätigen
    • Eintrag "Elexis 3 Base release" mit "Edit" bearbeiten, Name auf "Elexis 3 Base 3.5" und Location auf "http://download.elexis.info/elexis/3.5/p2/elexis-3-base" setzen. Mit "OK" bestätigen
    • Auf "Cancel" drücken
  • Elexis wird jetzt über "Hilfe..Auf Updates überprüfen" auf Elexis auf 3.5 hochgezogen werden

Upgrade Medelexis-Kunden

Der Upgrade wird von Medelexis mit Praxis abgesprochen und sollte danach ohne manuelle Anpassungen bei jeder Arbeitsstation beim nächsten Herunterfahren erfolgen. Bei Prolemen bitte bei Medelexis AG [Kontakt:https://medelexis.ch/?page_id=1061] melden

Arbeiten nach dem Upgrade

Nach einem Upgrade von 3.4.0 auf 3.5.0 sind keine weiteren Wartungsaufgaben notwendig.
