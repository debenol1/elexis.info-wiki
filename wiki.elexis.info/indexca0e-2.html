Auf dieser Seite wird erklärt, wie Sie eine Demoversion des Elexis herunterladen und installieren können.

Eine Anleitung für die normale Installation des Elexis finden sie unter Installationsanleitung Elexis

Herunterladen des Elexis

Öffnen Sie das Web Frontend und loggen Sie sich mit Ihrem Benutzername und Passwort ein. Wenn sie noch kein Login haben können Sie sich neu registrieren.



Im Menü „Allgemein“ (1) sehen Sie Ihre Kontaktdaten, die Sie bei Änderung jederzeit anpassen können. Haben Sie noch kein Login, können Sie sich jederzeit Registrieren.

Nach dem Registrieren auf dem Web Frontend erhalten Sie eine E-Mail zur Aktivierung, worin Ihnen auch Ihr Benutzername bekannt gegeben wird.


Über das Menü „Mein Medelexis“ (2) gelangen Sie zur Intallationsauswahl des Elexis.


Wählen Sie die entsprechende Datei aus (1) und laden Sie diese herunter (2).





Zip-Datei entpacken

Öffnen Sie den Ordner wo die Datei gespeichert wurde (3) und entpacken Sie die Datei mit rechtem Mausklick (4) an einem gewünschten Ort (5) mit einem Namen Ihrer Wahl (6).





Demo-Datenbank

Laden Sie sich die Demo-Datenbank hierherunter. sie finden die Datenbank auch auf unserer Homepage unter Elexis.


Entpacken Sie die Zip-Datei unter „Benutzer -> Ihr Benutzer -> elexis“. (1)



Installation

Öffnen Sie den Ordner wo das Elexis abgespeichert wurde. Mit einem Doppelklick auf „Medelexis“ (1) öffnen Sie das Programm.




Stimmen Sie der Lizenzvereinbarung zu (2) und richten Sie die Firewall-Einstellung (3) wie folgt ein:


  • Domänennetzwerk
  • Private Netzwerke


Anschliessen auf Zugriff zulassen klicken.




Anmeldung

In der Demo-Version müssen Sie sich mit folgenden Angaben einloggen.

  • Benutzername: test
  • Passwort: test
